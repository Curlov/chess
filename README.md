# Curlov' Chess Nova

A small chess web project featuring a JavaScript-based frontend and a chess engine written in Rust, compiled to WebAssembly (WASM) for fast execution in the browser.

## Tech Stack
- JavaScript (Frontend)
- Node.js
- Rust
- WebAssembly (WASM)

## Features
- Play chess in the browser
- Rust-based chess engine compiled to WASM
- Fast move calculation in the browser
